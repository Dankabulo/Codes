    Xamarin Forms 101: How to create a popup form in Xamarin Forms

    August 27, 2018 Bywebmasterdevlin InXamarin Forms

    Are you looking for an easy way to create a popup with forms?

    Would you like to create an awesome login modal page?

    In this post, we will create a popup page with entry views in Xamarin Forms.

    First thing’s first, here is the GitHub repo of this demo.

    https://github.com/webmasterdevlin/PopUpDemoApp

    Step 1: Right-click on your solution the select Manage NuGet Packages for Solution

     

    Step 2:  type rg.plugins.pop in the Browse section

     

    Step 3: Install the NuGet package in all projects.

     

    Go to your Android project -> MainActivity.cs and add Rg.Plugins.Popup.Popup.Init();

     

    Go to your iOS project -> AppDelegate.cs and add Rg.Plugins.Popup.Popup.Init();

     

    Step 4: Create a Content View in your NET Standard project. Name it PopTaskView

     

    Step 5: Add the xmlns namespace of Rg.Plugins.Popup library.

    Step 7:  Create two buttons. The 1st button will have the click event for popping up the modal page.

     

    Step 8: Create a method that will push a new PopupNewTaskView in the PopupNavigation.

     

    Step 9: Create the PopupPage. We’ll include also an animation for the PopupPage. We’ll also include a DatePicker and a TimePicker

     

    Step 10: Bind the Button and the Entry for disabling or enabling the button.
